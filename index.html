How it works

Navigate to https://bernardus.co.za/api/api/v1/generate/plain to generate some sick burns (birds).

It works in JSON too!

To grab them in JSON format, use https://bernardus.co.za/api/api/v1/generate/.

Adding words to the database

To add words, POST to /api/api/v1/add/<type>/<word>
